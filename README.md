## Features

- Object-oriented design for easy integration
- Comprehensive test suite with Pest
- Clear documentation for developers

## Requirements

- PHP 8.1 or higher


## Installation

Install the package via Composer:

```bash
composer require omisai/php-creditonline
```

## Usage

The package provides object-oriented access to the CreditOnline API. See [USAGE.md](USAGE.md) for detailed documentation and examples.


## Testing

Run the test suite using Pest:

```bash
composer test
```
